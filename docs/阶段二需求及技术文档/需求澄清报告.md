# 财务报表重构需求澄清报告

**文档版本**：v1.0  
**确认日期**：2025-03-04  
**状态**：已确认

---

## 1. 项目背景

### 1.1 当前状态
- 已完成A股上市公司财务数据采集，存储在本地SQLite数据库中
- 数据库包含5000+家公司的资产负债表、利润表、现金流量表数据
- 数据覆盖最近10年（季度数据）

### 1.2 需求来源
用户基于财务分析框架，设计了三张财务报表的重构方案，定义在Excel模板中：
- `资产资本表.xlsx` - 资产负债表重构
- `股权价值增加表.xlsx` - 利润表重构
- `现金流量分析.xlsx` - 现金流量表重构

### 1.3 项目目标
将Excel中定义的重构规则转化为代码实现，实现：
1. 从数据库读取指定公司的原始财务报表数据
2. 按照重构规则计算各项指标
3. 输出重构后的财务报表（Excel格式）
4. 为后续的汇总分析提供数据基础

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 数据读取 | 从SQLite数据库读取指定公司的三张财务报表 | P0 |

### 2.2 命令行接口

```bash
# 分析单个公司
python -m analysis.restructure --code 000333 --years 10 --output-dir output/restructure

# 批量分析多家公司
python -m analysis.restructure --codes 000333,000001,600000 --years 10 --output-dir output/restructure

# 输出示例
# output/restructure/000333_重构报表_2025-03-04.xlsx
# output/restructure/000001_重构报表_2025-03-04.xlsx

### 2.3 参数说明

## 3. 重构规则详细说明

### 3.1 资产资本表（资产负债表重构）

#### 3.1.1 重构目标
将传统资产负债表重构为"资产-资本"结构，区分金融资产、经营资产、有息债务、所有者权益。

#### 3.1.2 结构设计

```
资产结构
├── 金融资产合计
│   ├── 货币资金
│   ├── 拆出资金
│   ├── 交易性金融资产
│   ├── 衍生金融资产(流动)
│   ├── (其中)应收利息
│   ├── (其中)应收股利
│   ├── 买入返售金融资产
│   ├── 持有待售资产
│   ├── 一年内到期的非流动资产
│   ├── 发放贷款及垫款(流动)
│   ├── 发放贷款及垫款(非流动)
│   ├── 债权投资
│   ├── 其他债权投资
│   ├── 其他权益工具投资
│   ├── 其他非流动金融资产
│   └── 投资性房地产
├── 长期股权投资
└── 经营资产合计
    ├── 周转性经营投入合计 = 营运资产小计 - 营运负债小计
    │   ├── 营运资产小计
    │   │   ├── (其中)应收票据
    │   │   ├── (其中)应收账款
    │   │   ├── 应收款项融资
    │   │   ├── 预付款项
    │   │   ├── 其他应收款（扣除应收利息、应收股利）
    │   │   ├── 存货
    │   │   ├── 合同资产
    │   │   ├── 长期应收款
    │   │   └── 其他流动资产
    │   └── 营运负债小计
    │       ├── (其中)应付票据
    │       ├── (其中)应付账款
    │       ├── 预收账款
    │       ├── 合同负债
    │       ├── 应付职工薪酬
    │       ├── 长期应付职工薪酬
    │       ├── 应交税费
    │       ├── 其他应付款（扣除应付利息、应付股利）
    │       └── 其他流动负债
    └── 长期经营资产合计
        ├── 固定资产
        ├── 在建工程
        ├── 生产性生物资产
        ├── 公益性生物资产
        ├── 油气资产
        ├── 使用权资产
        ├── 无形资产
        ├── 开发支出
        ├── 商誉
        ├── 长期待摊费用
        ├── 其他非流动资产
        ├── 递延所得税资产
        └── 递延所得税负债

资本结构
├── 有息债务合计 = 短期债务 + 长期债务
│   ├── 短期债务
│   │   ├── 短期借款
│   │   ├── 向中央银行借款
│   │   ├── 拆入资金
│   │   ├── 交易性金融负债
│   │   ├── 衍生金融负债
│   │   ├── 卖出回购金融资产
│   │   ├── 吸收存款及同业存放
│   │   ├── 代理买卖证券款
│   │   ├── 代理承销证券款
│   │   ├── (其中)应付利息
│   │   ├── 持有待售负债
│   │   └── 一年内到期的非流动负债
│   └── 长期债务
│       ├── 长期借款
│       ├── 应付债券
│       ├── 租赁负债
│       └── 长期应付款
├── 所有者权益合计
│   ├── 归属于母公司股东权益合计
│   │   ├── 股本
│   │   ├── 其他权益工具
│   │   ├── (其中)优先股
│   │   ├── (其中)永续债
│   │   ├── 资本公积
│   │   ├── 减：库存股
│   │   ├── 其他综合收益
│   │   ├── 专项储备
│   │   ├── 盈余公积
│   │   ├── 一般风险准备金
│   │   └── 未分配利润
│   └── 少数股东权益
└── 资本总额 = 所有者权益 + 有息债务
```

#### 3.1.3 核心计算公式

| 项目 | 计算规则 |
|------|---------|
| 其他应收款 | = 原始其他应收款 - 应收利息 - 应收股利（需手动计算） |
| 其他应付款 | = 原始其他应付款 - 应付利息 - 应付股利（需手动计算） |
| 周转性经营投入合计 | = 营运资产小计 - 营运负债小计 |
| 经营资产合计 | = 周转性经营投入合计 + 长期经营资产合计 |
| 资产总额 | = 金融资产合计 + 长期股权投资 + 经营资产合计 |
| 有息债务合计 | = 短期债务 + 长期债务 |
| 所有者权益合计 | = 归属母公司股东权益 + 少数股东权益 |
| 资本总额 | = 所有者权益合计 + 有息债务合计 |

> **注意**：
> - 数据库中的 `other_receivables` 字段未扣除应收利息和应收股利，需手动计算
> - 递延所得税资产和递延所得税负债全部归入经营资产，不做经营性/金融性区分

#### 3.1.4 小计项目详细计算公式

**金融资产合计**：
```
= 货币资金 + 拆出资金 + 交易性金融资产 + 衍生金融资产(流动) + 应收利息 + 应收股利 
  + 买入返售金融资产 + 持有待售资产 + 一年内到期的非流动资产 + 发放贷款及垫款(流动) 
  + 发放贷款及垫款(非流动) + 债权投资 + 其他债权投资 + 其他权益工具投资 
  + 其他非流动金融资产 + 投资性房地产
```

**营运资产小计**：
```
= 应收票据 + 应收账款 + 应收款项融资 + 预付款项 + 其他应收款 + 存货 + 合同资产 
  + 长期应收款 + 其他流动资产 + 结算备付金 + 应收保费 + 应收分保账款 
  + 应收分保合同准备金 + 保险合同准备金
```

**营运负债小计**：
```
= 应付票据 + 应付账款 + 预收账款 + 合同负债 + 应付职工薪酬 + 长期应付职工薪酬 
  + 应交税费 + 其他应付款 + 应付手续费及佣金 + 应付分保账款 + 其他流动负债 
  + 预计负债(流动) + 预计负债(非流动) + 一年内到期的递延收益 + 长期递延收益 
  + 其他非流动负债
```

**长期经营资产合计**：
```
= 固定资产 + 在建工程 + 生产性生物资产 + 公益性生物资产 + 油气资产 + 使用权资产 
  + 无形资产 + 开发支出 + 商誉 + 长期待摊费用 + 其他非流动资产 + 递延所得税资产 
  - 递延所得税负债
```
> 注意：递延所得税负债作为减项，因为它本质上是负债而非资产

**短期债务**：
```
= 短期借款 + 向中央银行借款 + 拆入资金 + 交易性金融负债 + 衍生金融负债 
  + 卖出回购金融资产 + 吸收存款及同业存放 + 代理买卖证券款 + 代理承销证券款 
  + 应付利息 + 持有待售负债 + 一年内到期的非流动负债
```

**长期债务**：
```
= 长期借款 + 应付债券 + 租赁负债 + 长期应付款
```

**归属于母公司股东权益合计**：
```
= 股本 + 其他权益工具 + 资本公积 - 库存股 + 其他综合收益 + 专项储备 + 盈余公积 
  + 一般风险准备金 + 未分配利润
```

详见 `config/column_mapping.yaml` 中的 `balance_sheet_mapping`。

### 3.2 股权价值增加表（利润表重构）

#### 3.2.1 重构目标
将传统利润表重构为股权价值增加表，计算息税前经营利润、息前税后利润、股权价值增加值等核心指标。

#### 3.2.2 结构设计

```
股权价值增加表
├── 营业收入
├── 营业成本
├── 营业成本率 = 营业成本 / 营业收入
├── 毛利 = 营业收入 - 营业成本
├── 毛利率 = 毛利 / 营业收入
├── 税金及附加
├── 营业税金及附加率 = 税金及附加 / 营业收入
├── 销售费用
├── 销售费用率 = 销售费用 / 营业收入
├── 管理费用
├── 管理费用率 = 管理费用 / 营业收入
├── 研发费用
├── 研发费用率 = 研发费用 / 营业收入
├── 资产减值损失
├── 信用减值损失
├── 资产减值损失率 = (资产减值损失 + 信用减值损失) / 营业收入
├── 资产处置收益
├── 加：其他收益
├── 加：营业外收入
├── 减：营业外支出
├── 营业外收支及其他占营业收入的比例
├── 息税前经营利润 = 营业收入 - 营业成本 - 税金 - 销售费用 - 管理费用 - 研发费用 - 资产减值损失 + 其他收益 + 营业外收入 - 营业外支出
├── 经营利润所得税
├── 息前税后经营利润 = 息税前经营利润 - 经营利润所得税
├── 投资收益
│   ├── (其中)对联营企业及合营企业的投资收益
│   └── 短期投资收益 = 投资收益 - 对联营企业投资收益
├── (其中)利息收入
├── 净敞口套期收益
├── 公允价值变动收益
├── 汇兑收益
├── 其他综合收益的税后净额
├── 息税前金融资产收益 = 短期投资收益 + 利息收入 + 净敞口套期收益 + 公允价值变动收益 + 汇兑收益 + 其他综合收益
├── 金融资产收益所得税
├── 息前税后金融资产收益 = 息税前金融资产收益 - 金融资产收益所得税
├── 长期股权投资收益
├── 息税前利润总额 = 息前税后经营利润 + 息前税后金融资产收益 + 长期股权投资收益
├── 息前税后利润总额
├── 财务费用
├── 真实财务费用 = 财务费用 + 利息收入（加回被扣除的利息收入）
├── (其中)利息费用
├── 财务费用抵税效应
├── 税后真实财务费用 = 真实财务费用 - 财务费用抵税效应
├── 税前利润 = 息税前利润总额 - 真实财务费用
├── 减：所得税费用
├── 实际所得税税率 = 所得税费用 / 税前利润
├── 净利润 = 税前利润 - 所得税费用
├── 净利润率 = 净利润 / 营业收入
├── 股权资本成本（默认8%）
└── 股权价值增加值 = 净利润 - 股权资本成本
```

#### 3.2.3 核心计算公式

| 项目 | 计算规则 |
|------|---------|
| 毛利 | = 营业收入 - 营业成本 |
| 息税前经营利润 | = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 资产减值损失 - 信用减值损失 + 资产处置收益 + 其他收益 + 营业外收入 - 营业外支出 |
| **实际所得税税率** | = 所得税费用 / (税前利润 - 长期股权投资收益) |
| **经营利润所得税** | = 息税前经营利润 × 实际所得税税率 |
| 息前税后经营利润 | = 息税前经营利润 - 经营利润所得税 |
| 短期投资收益 | = 投资收益 - 对联营企业投资收益 |
| 息税前金融资产收益 | = 短期投资收益 + 利息收入 + 净敞口套期收益 + 公允价值变动收益 + 汇兑收益 + 其他综合收益 |
| **金融资产收益所得税** | = 息税前金融资产收益 × 实际所得税税率 |
| 息前税后金融资产收益 | = 息税前金融资产收益 - 金融资产收益所得税 |
| 息税前利润总额 | = 息前税后经营利润 + 息前税后金融资产收益 + 长期股权投资收益 |
| 真实财务费用 | = 财务费用 + 利息收入 |
| **财务费用抵税效应** | = 真实财务费用 × 实际所得税税率 |
| 税后真实财务费用 | = 真实财务费用 - 财务费用抵税效应 |
| 税前利润 | = 息税前利润总额 - 真实财务费用 |
| 净利润 | = 税前利润 - 所得税费用 |
| 股权资本成本 | 默认值：8%（可在配置文件中调整） |
| 股权价值增加值 | = 净利润 - 股权资本成本 |

#### 3.2.4 所得税项目计算说明

**设计原理**：将所得税按利润来源进行分配，分别计算经营利润、金融资产收益、财务费用各自承担的所得税影响。

| 项目 | 公式 | 说明 |
|------|------|------|
| **实际所得税税率** | = 所得税费用 / (税前利润 - 长期股权投资收益) | 排除长期股权投资收益（通常适用不同税率）后的实际税率 |
| **经营利润所得税** | = 息税前经营利润 × 实际所得税税率 | 经营活动产生的利润应承担的所得税 |
| **金融资产收益所得税** | = 息税前金融资产收益 × 实际所得税税率 | 金融资产投资收益应承担的所得税 |
| **财务费用抵税效应** | = 真实财务费用 × 实际所得税税率 | 利息支出可抵税，产生的抵税收益 |

**计算顺序**：
1. 先计算实际所得税税率
2. 分别计算三个所得税相关项目
3. 用所得税项目计算税后利润

**示例**（假设数据）：
```
息税前经营利润 = 100亿
息税前金融资产收益 = 10亿
真实财务费用 = 5亿
所得税费用 = 20亿
长期股权投资收益 = 2亿
税前利润 = 80亿

实际所得税税率 = 20 / (80 - 2) = 25.64%

经营利润所得税 = 100 × 25.64% = 25.64亿
金融资产收益所得税 = 10 × 25.64% = 2.56亿
财务费用抵税效应 = 5 × 25.64% = 1.28亿
```

### 3.3 现金流量分析表

#### 3.3.1 重构目标
分析自由现金流、资本支出、债务筹资等核心现金流指标。

#### 3.3.2 结构设计

```

### 3.4 TTM计算规则

#### 3.4.1 TTM定义
TTM（Trailing Twelve Months）= 滚动12个月数据，用于消除季节性波动影响。

#### 3.4.2 适用范围

| 报表类型 | 数据性质 | TTM计算 |
|---------|---------|---------|
| 资产负债表 | 时点数据 | **不需要**，直接取最新季度数据 |
| 利润表 | 期间数据（累计值） | **需要**，计算滚动12个月 |
| 现金流量表 | 期间数据（累计值） | **需要**，计算滚动12个月 |

#### 3.4.3 计算公式

**数据库数据特点**：季度数据为累计值
```
Q1 = 1-3月累计
Q2 = 1-6月累计
Q3 = 1-9月累计
Q4 = 1-12月累计（年度数据）
```

**TTM计算公式**：
```
TTM = 今年Q3累计 - 去年Q3累计 + 去年Q4累计
    = 今年Q3 + 去年Q4单季度
    = 最近12个月滚动数据
```

**示例**（2025Q3-TTM）：
```
TTM = 2025Q3累计 - 2024Q3累计 + 2024Q4累计
    = (2025Q1+Q2+Q3) - (2024Q1+Q2+Q3) + (2024Q1+Q2+Q3+Q4)
    = 2024Q4 + 2025Q1+Q2+Q3
    = 最近12个月
```

#### 3.4.4 实现逻辑

```python
def calculate_ttm(current_q3: float, last_year_q3: float, last_year_q4: float) -> float:
    """
    计算TTM值
    
    Args:
        current_q3: 今年Q3累计值
        last_year_q3: 去年Q3累计值
        last_year_q4: 去年Q4累计值（年度数据）
    
    Returns:
        TTM值
    """
    return current_q3 - last_year_q3 + last_year_q4
```

#### 3.4.5 异常处理规则

| 异常情况 | 处理方式 |
|---------|---------|
| 数据缺失（缺少去年Q3或Q4数据） | 标记为无效，输出 "-" |
| 新股/次新股（上市不足一年） | 标记为无效，输出 "-" |
| 历史报告期数据缺失 | 使用已有年度数据，不计算该年度TTM |

> **TTM计算范围**：只计算最新报告期的TTM，历史年度数据使用完整年报数据

## 4. 输出格式

### 4.1 文件格式
Excel格式（.xlsx），便于后续整合到汇总分析表。

```
{股票代码}_重构报表_{生成日期}.xlsx
示例：000333_重构报表_2025-03-04.xlsx
```

> 日期为文件生成日期，非报告期日期

### 4.3 Sheet结构

| Sheet名称 | 内容 |
|----------|------|
| 资产资本表 | 资产负债表重构结果 |
| 股权价值增加表 | 利润表重构结果 |
| 现金流量分析表 | 现金流量表重构结果 |

### 4.4 数据格式

每个Sheet的数据格式：

| 行 | 内容 |
|----|------|
| 第1行 | 报告日期列标题（如：项目、2025Q3-TTM、2024-12-31、2023-12-31...） |
| 第2行 | 单位/备注 |
| 第3行起 | 重构项目数据 |

**列顺序**：
1. A列：项目名称
2. B列：TTM数据（最新）
3. C列起：年度数据（倒序，最近10年）

### 4.5 数据规范

| 规范项 | 说明 |
|--------|------|
| 数据单位 | 统一采用数据库中的原始单位（元） |
| 百分比格式 | 输出为百分比形式（如25%而非0.25） |
| 空值表示 | 数据缺失时输出 "-" |
| 中间计算过程 | 需要输出所有小计项目（如营运资产小计、营运负债小计等） |
